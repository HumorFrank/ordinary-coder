# VSCode插件

## VSCode Marketplace

> [VSCode Marketplace](https://marketplace.visualstudio.com/VSCode): Visual Studio Code 扩展

## VSCode 翻译

### `Chinese (Simplified)`

- 作者：Microsoft
- 功能：适用于 VS Code 的中文（简体）语言包

## 代码质量与格式化

### `ESLint`

- 作者：Microsoft
- 功能：将ESLint JavaScript集成到VS Code中。

### `Prettier-Code formatter`

- 作者：Prettier
- 功能：使用prettier的代码格式化工具

### Auto Close Tag

- 作者：Jun Han
- 功能：自动添加HTML/XML闭合标签，与Visual Studio IDE或Sublime Text相同

### Auto Complete Tag

- 作者：Jun Han
- 功能：扩展包用于为HTML/XML自动添加闭合标签并重命名配对标签

### Auto Rename Tag

- 作者：Jun Han
- 功能：自动重命名成对的HTML/XML标签

## 调试与运行

### JavaScript Debugger(Nightly)

- 作者：Microsoft
- 功能：用于调试 Node.js 程序和 Chrome 的扩展程序。

### Vite

- 作者：Anthony Fu
- 功能：VS Code for Vite

### Live Server

- 作者：Ritwick Dey
- 功能：启动具备实时刷新功能的本地开发服务器，支持静态与动态页面

## 辅助与美化

### `Path Intellisense`

- 作者：Christian Kohler
- 功能：自动补全文件路径，支持 `@/` 等别名解析。

### `Auto Import`

- 作者：steoates
- 功能：自动查找、解析并为所有可用导入项提供代码操作和代码补全功能。兼容TypeScript和TSX。

### Vue Theme
- 作者：mariorodeghiero
- 功能：Visual Studio Code 主题灵感源自 JavaScript，支持更多主流语言，色彩搭配完美协调。

## Vue

### `Vue VSCode Snippets`

- 作者：sarah.drasner
- 功能：Snippets that will supercharge your Vue workflow

### Vue 3 Snippets

- 作者：hollowtree
- 功能：Vue.js 3 与 Vue.js 2 代码片段扩展

### `Vue (Official)`

- 作者：官方
- 功能：Language Support for Vue

### Vue Peek

- 作者：Dario Fuzinato
- 功能：支持查看和跳转至Vue单文件组件的定义。

## Git

### `Git History`

- 作者：Don Jayamanne
- 功能：查看Git日志、文件历史记录、比较分支或提交

### `GitLens`

- 作者：GitKraken
- 功能：可视化代码作者信息
- 缺点：pro 版本收费

### `Git Graph`

- 作用：mhutchie
- 功能：查看您存储库的 Git 图，并从图中执行 Git 操作。

## 注释翻译

### `Comment Translate`

- 作者：intellsmi
- 功能：帮助开发人员翻译代码中的注释、字符串、代码提示、错误消息和变量名称。

## 注释标签

### `Todo Tree`

- 作者：Gruntfuggly
- 功能：在树形视图中显示 TODO、FIXME 等注释标签
