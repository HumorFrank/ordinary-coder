# VSCode插件

## VSCode Marketplace
> [VSCode Marketplace](https://marketplace.visualstudio.com/VSCode): Visual Studio Code 扩展

## Vue

- `Vue (Official)`
  - 作者：官方
  - 功能：Language Support for Vue
- `Vite`
  - 作者：Anthony Fu
  - 功能：VS Code for Vite

## Git

- `Git History`
  - 作者：Don Jayamanne
  - 功能：查看Git日志、文件历史记录、比较分支或提交
- `GitLens`
  - 作者：GitKraken
  - 功能：可视化代码作者信息
  - 缺点：pro 版本收费
- `Git Graph`
  - 作用：mhutchie
  - 功能：查看您存储库的 Git 图，并从图中执行 Git 操作。

## 注释翻译

- `Comment Translate`
  - 作者：intellsmi
  - 功能：帮助开发人员翻译代码中的注释、字符串、代码提示、错误消息和变量名称。

## 注释标签

- `Todo Tree`
  - 作者：Gruntfuggly
  - 功能：在树形视图中显示 TODO、FIXME 等注释标签
